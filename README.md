### Accessing the challenge in Docker
## Run commands in the terminal-


Go to the challenge directory:
```bash 
cd challenge1
``` 

Run the docker container: 
```bash 
npm start
```

Then: 
```
docker compose up
```
Then navigate to:
```bash
http://localhost:5001/Challenge2
```

Exit and delete the docker container:
```bash
docker compose down
```
