### Accessing the challenges in Docker
## Run commands in the terminal-

Run the docker container: 
```bash 
docker compose up
```
Then navigate to:
```bash
http://localhost:8000/challenge1
```

Exit and delete the docker container:
```bash
docker compose down
```
